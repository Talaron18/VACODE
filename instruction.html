Welcome to VACODE

System Requirements

To use VACODE successfully, you need to have the following:
-- C:\\Windows\\System32\\WindowsPowerShell\\v1.0\\powershell.exe
-- mingw64 (to compile C/C++ code)

Note:the mingw64 has to be installed in the following directory: C:\mingw64

User Guide

1.File operations: Click on the "File" button to create or open files or floders.

Note: the default directory of created file or folder is C:\Users\YourUserName\Desktop.
on creating a new file or folder in the default directory, you may not see it in the
explorer. Please click the "open file" or "open folder" button to open it.

2.Code editor: Click on the file you want to edit to open the code editor.
3.Terminal: the integrated terminal is automatically opened when you open this application.

*the working directory of the terminal is C:\Users\YourUserName. You need to manually
change it to the directory you want to work in with the "cd" command.

4.Run: the code you opened is interpreted as the code you want to run. Click the "Run" button on the left to compile and run.

Note: the run button currently only supports Python/C/C++ code.

5.AI: click the "AI" button on the left to open the AI assistant. You can ask questions about coding and get answers from the AI.

*the AI assistant is powered by qwen3-8b(siliconflow) and may require a few minutes to return an answer.
